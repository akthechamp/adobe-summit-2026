# External Contributions Archive

This folder contains brand assets contributed by external collaborators that **do not match the locked AI Forte brand system**. They are preserved for historical reference but should not be used.

## Why these are archived

The locked brand system (`resources/brand-system.md`) specifies:
- **Palette A — Warm Premium** as the only approved palette: cream `#F8F3EC`, deep espresso `#2C1810`, burnt sienna `#C65D2E`, warm taupe `#8B7355`, cream border `#E8DDD0`
- **Editorial premium aesthetic** — Aesop, Kinfolk, Wallpaper magazine references
- **No dark-mode-AI-network iconography** — explicitly rejected (see `resources/palette-samples/DECISION.md`)
- **No stock AI clichés** — robot handshakes, brain icons, glowing neural nets, etc. are explicitly banned

The locked decisions were documented Apr 14, 2026 (commit `5ec0b1a`).

## How to prevent future drift

If you're a contributor reading this:
1. **Read `resources/brand-system.md` first.** Every brand asset must use Palette A and follow the editorial premium aesthetic specified there.
2. **Read `resources/palette-samples/DECISION.md`** for the rationale behind why dark/violet/network-node aesthetics were specifically rejected.
3. **Check `CHANGELOG.md`** for the latest brand decisions before generating assets.
4. **If you want to propose a brand pivot, open a discussion first** — don't ship assets in a different brand system to main.
