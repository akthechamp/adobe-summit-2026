# Headshot — Background Replacement

Your current photo (`ashish-pic.jpg`) is a great casual shot at the Golden Gate Bridge — smiling, confident, clear face. For LinkedIn and professional collateral, you have two options:

## Option 1: Use Gemini 3 to Replace Background

Upload `ashish-pic.jpg` to Gemini and use this prompt:

```
Edit this photo: replace the background with a clean, professional neutral-gradient
backdrop suitable for a B2B tech executive LinkedIn profile photo.

Use a soft gradient in subtle navy/charcoal tones (#1A2332 to #2C3548) — similar
to high-end SaaS founder headshots. Keep all facial features, expression, lighting
on the face, hair, and sunglasses exactly unchanged. Add very subtle rim lighting
on the shoulders consistent with the new background.

Output: clean, sharp, editorial quality headshot. Maintain original image resolution.
Crop to head-and-shoulders composition if the current framing is wider.
```

**Alternative prompt (lighter background):**
```
Edit this photo: replace the background with a soft, blurred light grey-white
gradient suitable for a professional LinkedIn headshot. Keep the subject
(face, expression, clothing, sunglasses) completely unchanged. Add subtle
professional studio lighting wrap on the edges of the shoulders.

Clean, sharp, modern professional headshot aesthetic.
```

## Option 2: Use as-is

The Golden Gate Bridge photo is actually a strong LinkedIn photo — it's authentic, you look confident and approachable, and the setting is memorable. Many founders use casual-but-high-quality photos. Consider using it without modification.

## Recommendation

For Summit specifically: use the original photo as-is on LinkedIn (it's authentic and memorable). Generate the professional-background version for the one-pager PDF and digital business card where a more formal look is expected.
